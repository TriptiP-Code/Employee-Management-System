# Employee Management System (EMS)

## Overview

Employee Management System (EMS) is a role-based task management application built with React.js. The application allows administrators to create and assign tasks to employees, while employees can view and manage their assigned work through a personalized dashboard.

The project demonstrates modern React development practices including Context API for state management, authentication handling, localStorage-based persistence, reusable component architecture, and role-based access control.

## Features

### Authentication & Authorization

* Login functionality for Admin and Employees
* Role-based dashboard rendering
* Persistent login sessions using localStorage
* Secure logout functionality

### Admin Dashboard

* View employee information
* Create and assign tasks to employees
* Track task status and progress
* Manage employee task distribution

### Employee Dashboard

* View assigned tasks
* Track task progress
* Manage task statuses
* Personalized user experience based on login credentials

### Data Management

* Context API for global state management
* LocalStorage integration for persistent data storage
* Dynamic task creation and updates
* Real-time UI updates based on user actions
